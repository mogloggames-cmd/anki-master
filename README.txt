Tax Tutor - 税理士試験 理論暗記管理アプリ
バージョン: 1.0
作成日: 2026年1月17日
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【アプリの概要】
税理士試験（財務諸表論など）の理論暗記に特化した学習管理アプリです。
問題文と解答を同時表示する暗記カード方式と、累積型復習管理システムを
組み合わせた効率的な学習環境を提供します。

【主な機能】
✅ 問題文・解答同時表示の暗記カード方式
✅ SABC評価システムによる復習間隔の自動管理
✅ 穴埋め型・説明型の2つの問題形式に対応
✅ 4種類の穴埋め入力記法（_, "", 全角スペース, ___）
✅ 個別登録・一括登録の両対応
✅ 教材構造管理（科目・問題集・単元の編集・削除）
✅ データのエクスポート/インポート機能
✅ 負荷予測カレンダー
✅ 完全オフライン動作（インターネット不要）

【インストール方法】

■ 方法1：直接開く（最も簡単）
1. ZIPファイルを解凍
2. tax_tutor_app フォルダ内の index.html をダブルクリック
3. ブラウザで開く → すぐに使えます！

■ 方法2：ローカルサーバーで開く（推奨）
1. ZIPファイルを解凍
2. tax_tutor_app フォルダで右クリック → ターミナル/コマンドプロンプトを開く
3. 以下のコマンドを実行：
   
   Windows:
   python -m http.server 8080
   
   Mac/Linux:
   python3 -m http.server 8080

4. ブラウザで http://localhost:8080 を開く

【複数PC間でのデータ同期方法】

デスクトップPCとノートPCの両方で使う場合：

■ 初回セットアップ
1. 両方のPCに tax_tutor_app フォルダをコピー
2. 両方のPCで index.html を開いて動作確認

■ 日常の同期作業

【デスクトップPC（メインPC）】
1. 理論を登録・復習
2. ⚙️ 教材管理 → バックアップ
3. 📤 データをエクスポート
4. JSONファイルをGoogleドライブ/Dropbox等にアップロード

【ノートPC（サブPC）】
1. クラウドからJSONファイルをダウンロード
2. ⚙️ 教材管理 → バックアップ
3. 📥 データをインポート（完全置き換え）
4. ✅ メインPCと同じデータで学習開始！

【使い方ガイド】

■ タブ1：📅 今日の復習プール
- 今日復習すべき理論が自動的に表示されます
- 問題文と解答を見ながら暗記
- S・A・B・Cの評価ボタンで記録
- 評価に応じて次回復習日が自動設定されます

■ タブ2：📝 高速暗記記録
- 科目・問題集・単元を選択して集中学習
- 特定の単元を繰り返し暗記するのに最適

■ タブ3：📋 全理論一覧
- すべての理論を評価別・形式別に表示
- 編集・削除機能で問題を修正できます

■ タブ4：📅 負荷予測カレンダー
- 今後の復習予定を月間カレンダーで確認
- 日付をクリックすると、その日の復習リストが表示されます

■ タブ5：⚙️ 教材管理
- 個別登録：1問ずつ丁寧に登録
- 一括登録：テキスト形式で大量登録
- 教材構造管理：科目・問題集・単元の編集・削除
- バックアップ：エクスポート/インポート機能

【評価システム（SABC）】

S評価：完全定着（2週間以上経ってもスラスラ）
  → 次回復習：14日後

A評価：記憶安定（2回目以降で何も見ずスラスラ）
  → 次回復習：7日後

B評価：記憶不安定（2回目以降だが詰まる、回答見ながら暗唱）
  → 次回復習：翌日

C評価：初回接触（今日初めて触れた理論）
  → 次回復習：翌日

【穴埋め問題の入力方法】

以下の4つの記法がすべて使えます：

1. _ （アンダースコア1つ）← 最推奨
   例：企業会計は、_の報告を旨とすべきである。

2. "" （ダブルクォーテーション2つ）
   例：企業会計は、""の報告を旨とすべきである。

3. 全角スペース ← 最速入力
   例：企業会計は、　の報告を旨とすべきである。

4. ___ （アンダースコア3つ以上）← 従来方式
   例：企業会計は、___の報告を旨とすべきである。

※ どの記法も同じように穴埋め箇所として認識されます

【注意事項】

⚠️ データ保存について
- データはブラウザのlocalStorageに保存されます
- ブラウザのキャッシュをクリアするとデータが消えます
- 定期的にエクスポートしてバックアップを取りましょう

⚠️ 異なるブラウザ間でのデータ
- Chrome と Edge は別々のlocalStorageを持ちます
- 同じブラウザを使い続けることをお勧めします
- 別のブラウザで使う場合は、エクスポート/インポートで移行してください

⚠️ ファイルの移動
- アプリファイル（HTML/CSS/JS）を移動してもデータは移動しません
- データはlocalStorageに保存されているためです
- データを移すには、エクスポート/インポート機能を使ってください

【トラブルシューティング】

Q: データが消えてしまいました
A: クラウドに保存したバックアップファイル（JSON）をインポートしてください

Q: 別のPCでデータを使いたい
A: メインPCでエクスポート → JSONファイルをコピー → 別PCでインポート

Q: ブラウザを変えたらデータが見えなくなった
A: localStorageはブラウザごとに独立しています。元のブラウザで開いてください

Q: 間違えて登録した問題を削除したい
A: 📋 全理論一覧 または 📝 高速暗記 タブで「🗑️ 削除」ボタンを使ってください

Q: 科目名や問題集名を変更したい
A: ⚙️ 教材管理 → 教材構造管理 で「✏️ 編集」ボタンを使ってください
