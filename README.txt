━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【アプリの概要】
暗記学習を支援するフラッシュカードアプリです。
科学的な復習スケジュール（スペーシング効果）と直感的なUIで、
資格試験・語学・学校の試験など、あらゆる暗記学習をサポートします。

【主な機能】
- 今日の復習：当日やるべきカードを自動で表示
- テストモード：範囲を指定して実力チェック
- 不正解復習：間違えた問題だけを再確認
- 学習統計：日別学習量・正答率・定着率をグラフ表示
- 負荷予測カレンダー：今後の復習スケジュールを可視化
- 教材管理：科目・問題集・単元の階層構造で整理
- クラウド同期：複数デバイスでデータを共有
- PWA対応：ホーム画面に追加してアプリのように使用可能
- 完全オフライン動作対応

【評価システム】
E(翌日) → D(3日後) → C(7日後) → B(14日後) → A(30日後) → S(完全定着)
正解で1段階アップ、不正解時はモード(ストイック/ノーマル/やさしい)で変動
